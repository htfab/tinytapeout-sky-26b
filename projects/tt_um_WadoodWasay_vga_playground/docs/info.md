## How it works

It displays stripes in VGA output. This is a very slightly tweaked project from VGA playground, followed along with the Latch-Up 2026 TinyTapeout workshop tutorial. This isn't a project meant to be extensively used.  

## How to test

Hook it up to a VGA monitor. 

## External hardware

This project will require the VGA board PMOD. This project is providing two bits per color channel.
